Rules for writing this SOP:
- Use checkbox format for every single check so technicians can physically mark completion
- Every check must include what normal looks like AND exactly what to do if abnormal
- Include specific readings, temperatures, pressures, and values where available from the source data
- If a specific value is not in the source data write [VERIFY NORMAL RANGE WITH ENGINEER]
- Group checks by physical location so the technician walks the facility efficiently without backtracking
- Never use vague language like check the system — always specify exactly what to look at and what to look for
- Keep language simple and direct — written for a technician not an engineer
- Never assume the technician knows what normal looks like — always state it explicitly
